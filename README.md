# compiled_license
Compiles the licences of all shards used in project into binary.

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  compiled_license:
    github: elorest/compiled_license
```

## Usage

```crystal
require "compiled_license"
```

```
$ testapp --licenses
# Returns all licenses to stdout
```
